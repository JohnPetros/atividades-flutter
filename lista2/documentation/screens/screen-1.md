# Prompt — Tela Inicial / Menu de Exercícios

Create a **mobile home screen UI** for a Flutter educational app called **Flutter Lab**, designed as a **components laboratory** for students.
This screen is the **main menu of practical exercises**, where each item navigates to a different exercise screen.

## Language requirement

**All visible text in the interface must be in Brazilian Portuguese (pt-BR).**

## Screen purpose

Design the **home/menu screen** of the app, presenting a list of practical Flutter exercises based on interface components such as **TextField, Checkbox, Radio, Autocomplete, Calendar, ListView, and TabBarView**.

## Visual style

* Dark mode interface
* Elegant, modern, minimalistic mobile UI
* Premium educational app feel
* Clean layout with strong spacing and alignment
* Material Design 3 inspired
* Rounded cards with soft borders
* Subtle shadows and contrast
* White titles with gray secondary text
* Accent colors per exercise card
* Sleek and polished, like a real production Flutter app

## Header area

At the top, create a compact dark header with:

* App name: **Flutter Lab**
* Subtitle: **Laboratório de Componentes**
* A circular profile avatar on the right with the initials **JS**
* Strong typographic hierarchy
* Clean and balanced spacing

## Main content

Below the header, add the section title:

* **Exercícios Práticos**

Then create a **vertical list of large rounded cards**, one for each exercise.
Each card should include:

* a colored vertical accent bar on the left edge
* a dark icon container with rounded corners
* one icon related to the exercise
* exercise title in bold
* short descriptive subtitle in smaller muted text
* a chevron icon on the right indicating navigation
* premium spacing and card padding

## Layout behavior

* Vertical scroll screen
* Comfortable side margins
* Consistent spacing between cards
* Cards should feel tappable and interactive
* Use a smartphone aspect ratio
* Design should look ready to be implemented in Flutter

## Footer

At the bottom, add:

* a subtle Flutter-style icon or abstract mark
* the text: **Desenvolvido para fins didáticos**
* low emphasis, centered, elegant
